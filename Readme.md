Strona Ticket Hero która na celu ma sprzedawanie biletów na koncerty 

### User story - 1 - Wyszukiwanie Koncertów
Użytkownik może wyszukiwać koncerty za pomocą wielu filtrów, takichjak zakres cen, gatunek muzyki, artysta i inne.
### User story - 2 - Kupno Biletu
Po znalezieniu koncertu którym użytkownik jest zainteresowany, użytkownik może kupić bilet na stronie
### User story - 3 - Logowanie
Użytkwonik może się zalogować
### User story - 4 - Koncerty w Okolicy
Jeśli użytkownik podał swoje miejsce zamieszkania, system wyszuka i poda mu koncerty które odbędą się w jego okolicy.
